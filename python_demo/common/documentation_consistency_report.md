## 格式统一性检查

### ✅ 统一的格式标准

#### 1. 标题层级
- 一级标题：# （文档标题）
- 二级标题：## （主要章节）
- 三级标题：### （子章节）
- 四级标题：#### （详细内容）

#### 2. 代码块格式
```python
# Python 代码使用 python 语法高亮
```

```bash
# 命令行使用 bash 语法高亮
```

```yaml
# YAML 配置使用 yaml 语法高亮
```

#### 3. 列表格式
- 无序列表使用 `-` 
- 有序列表使用 `1.`
- 任务列表使用 `- [ ]` 和 `- [x]`

#### 4. 链接格式
- 内部链接：`[文本](相对路径)`
- 外部链接：`[文本](完整URL)`
- 参考链接：`[文本][ref]` + `[ref]: URL`

#### 5. 表格格式
```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 内容 | 内容 | 内容 |
```

#### 6. 强调格式
- **粗体**：`**文本**`
- *斜体*：`*文本*`
- `代码`：`` `代码` ``

#### 7. 引用格式
> 引用内容使用 `>` 符号

#### 8. 分隔线
使用 `---` 作为分隔线
