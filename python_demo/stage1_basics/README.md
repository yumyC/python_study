# 第一阶段：Python 基础

## 阶段简介

欢迎来到 Python 学习课程的第一阶段！本阶段将帮助你从零开始，掌握 Python 编程的基础知识，为后续的 Web 开发学习打下坚实的基础。

## 学习目标

完成本阶段学习后，你将能够：

- ✅ 搭建 Python 开发环境（PyCharm 或 VSCode）
- ✅ 理解并使用 Python 的基本数据类型
- ✅ 掌握控制流语句（if/for/while）
- ✅ 编写和使用函数
- ✅ 理解面向对象编程的基本概念
- ✅ 使用模块和包组织代码
- ✅ 进行文件读写操作
- ✅ 完成3个实践项目

## 前置知识

- 无需编程经验
- 基本的计算机操作能力
- 对编程有兴趣和学习热情

## 学习内容

### 1. 环境搭建 (setup/)

在开始编程之前，需要先搭建开发环境：

- [Python 安装指南](setup/python_installation.md) - 在 Windows/macOS/Linux 上安装 Python
- [PyCharm 虚拟环境设置](setup/pycharm_setup.md) - 使用 PyCharm IDE
- [VSCode 虚拟环境设置](setup/vscode_setup.md) - 使用 VSCode 编辑器
- [虚拟环境使用指南](setup/virtual_env_guide.md) - 理解和使用虚拟环境

**建议**: 选择一个你喜欢的编辑器（PyCharm 或 VSCode），完成环境搭建后再继续学习。

### 2. 基础教程 (tutorials/)

按顺序学习以下教程，每个教程都包含详细的代码示例和注释：

1. [变量和数据类型](tutorials/01_variables_and_types.py)
   - 变量定义和命名规则
   - 数字、字符串、布尔类型
   - 列表、元组、字典、集合
   - 类型转换

2. [控制流](tutorials/02_control_flow.py)
   - if/elif/else 条件语句
   - for 循环和 while 循环
   - break 和 continue
   - 列表推导式

3. [函数](tutorials/03_functions.py)
   - 函数定义和调用
   - 参数类型（位置、默认、可变参数）
   - 返回值和作用域
   - Lambda 函数和装饰器

4. [类和对象](tutorials/04_classes_and_objects.py)
   - 类的定义和实例化
   - 属性和方法
   - 继承和多态
   - 特殊方法

5. [模块和包](tutorials/05_modules_and_packages.py)
   - 模块的导入和使用
   - 创建自定义模块
   - 常用标准库
   - 包的组织

6. [文件操作](tutorials/06_file_operations.py)
   - 文件的读写
   - 目录操作
   - CSV 和 JSON 文件
   - 异常处理

**学习建议**:
- 每个教程都要亲自运行代码
- 尝试修改代码，观察结果
- 完成教程中的练习
- 不理解的地方多查资料

### 3. 实践项目 (projects/)

通过实际项目巩固所学知识：

1. [计算器](projects/project1_calculator/)
   - 基本四则运算
   - 高级数学运算
   - 历史记录功能
   - 错误处理

2. [待办事项列表](projects/project2_todo_list/)
   - 任务管理（增删改查）
   - 数据持久化（JSON）
   - 优先级和状态管理
   - 统计功能

3. [文件管理器](projects/project3_file_manager/)
   - 文件和目录操作
   - 文件信息查看
   - 文件搜索
   - 复制和移动

**项目建议**:
- 先理解项目需求
- 阅读代码，理解实现思路
- 尝试运行项目
- 根据扩展练习添加新功能

## 学习时间规划

建议学习时间：**2-4 周**

- 环境搭建：1-2 天
- 基础教程：1-2 周（每天 2-3 小时）
- 实践项目：1 周（每个项目 2-3 天）
- 复习巩固：3-5 天

**注意**: 学习时间因人而异，重要的是理解概念，而不是赶进度。

## 学习方法建议

### 1. 主动学习
- 不要只看代码，要亲自敲一遍
- 尝试修改代码，观察结果
- 遇到问题先思考，再查资料

### 2. 循序渐进
- 按顺序学习教程
- 理解一个概念后再继续
- 不要跳过基础内容

### 3. 实践为主
- 完成所有示例代码
- 做完三个实践项目
- 尝试自己的小项目

### 4. 记录笔记
- 记录重要概念
- 整理常见错误
- 总结学习心得

### 5. 寻求帮助
- 查阅官方文档
- 搜索相关问题
- 参与社区讨论

## 常见问题

### Q: 我完全没有编程经验，能学会吗？
A: 可以！本课程从零开始，循序渐进。只要认真学习，多加练习，一定能掌握。

### Q: 需要多长时间完成第一阶段？
A: 建议 2-4 周，但因人而异。重要的是理解概念，不要急于求成。

### Q: PyCharm 和 VSCode 选哪个？
A: 都可以。PyCharm 功能更强大，VSCode 更轻量。建议初学者使用 PyCharm Community 版。

### Q: 遇到错误怎么办？
A: 1) 仔细阅读错误信息 2) 检查代码拼写和语法 3) 搜索错误信息 4) 查看故障排除指南

### Q: 需要记住所有语法吗？
A: 不需要。理解概念更重要，具体语法可以随时查阅文档。
