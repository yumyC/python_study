# CRM 系统用户手册

## 1. 系统简介

CRM（客户关系管理）系统是一个企业级的员工和组织管理平台，提供员工管理、岗位管理、角色权限管理和工作日志等功能。

## 3. 功能模块

### 3.1 员工管理

**功能**: 管理企业员工信息

**操作步骤**:

1. **查看员工列表**
   - 点击左侧菜单"员工管理"
   - 查看所有员工信息
   - 使用搜索框搜索员工

2. **创建员工**
   - 点击"新增员工"按钮
   - 填写员工信息（用户名、邮箱、姓名、岗位、角色）
   - 点击"确定"保存

3. **编辑员工**
   - 点击员工列表中的"编辑"按钮
   - 修改员工信息
   - 点击"确定"保存

4. **删除员工**
   - 点击员工列表中的"删除"按钮
   - 确认删除操作

**权限要求**: HR 管理员或系统管理员

### 3.2 岗位管理

**功能**: 管理企业组织架构和岗位

**操作步骤**:

1. **查看岗位树**
   - 点击左侧菜单"岗位管理"
   - 查看岗位树形结构

2. **创建岗位**
   - 点击"新增岗位"按钮
   - 填写岗位信息（名称、编码、层级、上级岗位）
   - 点击"确定"保存

3. **编辑岗位**
   - 点击岗位列表中的"编辑"按钮
   - 修改岗位信息
   - 点击"确定"保存

4. **删除岗位**
   - 点击岗位列表中的"删除"按钮
   - 确认删除操作
   - **注意**: 有员工的岗位不能删除

**权限要求**: HR 管理员或系统管理员

### 3.3 角色管理

**功能**: 管理系统角色

**操作步骤**:

1. **查看角色列表**
   - 点击左侧菜单"角色管理"
   - 查看所有角色

2. **创建角色**
   - 点击"新增角色"按钮
   - 填写角色信息（名称、编码、描述）
   - 点击"确定"保存

3. **分配权限**
   - 点击角色列表中的"权限设置"按钮
   - 勾选菜单和操作权限
   - 点击"确定"保存

**权限要求**: 系统管理员

### 3.4 菜单管理

**功能**: 管理系统菜单和路由

**操作步骤**:

1. **查看菜单树**
   - 点击左侧菜单"菜单管理"
   - 查看菜单树形结构

2. **创建菜单**
   - 点击"新增菜单"按钮
   - 填写菜单信息（名称、路径、图标、组件）
   - 点击"确定"保存

**权限要求**: 系统管理员

### 3.5 工作日志

**功能**: 记录和管理员工日常工作

**操作步骤**:

1. **创建工作日志**
   - 点击左侧菜单"工作日志"
   - 点击"新增日志"按钮
   - 填写工作内容、完成情况、遇到的问题、明日计划
   - 进行自我评分（1-5 分）
   - 点击"确定"保存

2. **查看工作日志**
   - 查看自己的工作日志列表
   - 点击日志查看详情

3. **编辑工作日志**
   - 点击日志列表中的"编辑"按钮
   - 修改日志内容
   - **注意**: 只能编辑当天的日志

4. **上级评分**（主管功能）
   - 查看下属的工作日志
   - 点击"评分"按钮
   - 填写评分（1-5 分）和评语
   - 点击"确定"保存

5. **导出工作日志**
   - 点击"导出"按钮
   - 选择日期范围
   - 点击"确定"
   - 等待导出完成
   - 点击"下载"按钮下载 Excel 文件

**权限要求**: 
- 普通员工: 可以创建和查看自己的日志
- 部门主管: 可以查看和评分下属的日志
- 管理员: 可以查看所有日志

## 4. 个人设置

### 4.1 修改密码

1. 点击右上角用户头像
2. 选择"修改密码"
3. 输入旧密码
4. 输入新密码（至少 8 位，包含大小写字母和数字）
5. 确认新密码
6. 点击"确定"保存

### 4.2 查看个人信息

1. 点击右上角用户头像
2. 选择"个人信息"
3. 查看个人基本信息、岗位和角色

## 5. 常见问题

### Q1: 忘记密码怎么办？

A: 联系系统管理员重置密码。

### Q2: 为什么看不到某些菜单？

A: 可能是权限不足，联系管理员分配权限。

### Q3: 工作日志可以修改吗？

A: 只能修改当天的日志，已被上级评分的日志不能修改工作内容。

### Q4: 如何导出工作日志？

A: 在工作日志页面点击"导出"按钮，选择日期范围后等待导出完成。

### Q5: 导出的文件在哪里下载？

A: 导出完成后会显示下载链接，点击即可下载。下载链接 24 小时有效。

## 7. 安全建议

1. **密码安全**
   - 使用强密码（至少 8 位，包含大小写字母、数字）
   - 定期修改密码
   - 不要与他人共享密码

2. **账号安全**
   - 不要在公共电脑上保存密码
   - 使用完毕后及时退出登录
   - 发现异常登录及时联系管理员

3. **数据安全**
   - 不要泄露敏感信息
   - 定期备份重要数据
