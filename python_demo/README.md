# Python 学习课程体系

## 课程简介

欢迎来到 Python 学习课程体系！这是一套完整的、从零基础到企业级项目实战的 Python Web 开发课程。本课程采用渐进式学习路径，通过五个递进阶段，帮助你系统掌握 Python Web 开发的核心技能。

无论你是编程新手，还是想要提升 Python Web 开发能力的开发者，这套课程都能为你提供清晰的学习路径和丰富的实践项目。

## 课程特色

- **渐进式学习路径**: 从基础到进阶，循序渐进，降低学习曲线
- **双框架教学**: 同时学习 FastAPI 和 Flask，掌握主流 Web 框架
- **企业级实战**: 完整的 CRM 项目，涵盖认证、权限、异步任务等企业特性
- **工程化思维**: 包含测试、CI/CD、Docker 等现代开发实践
- **丰富的示例**: 每个知识点都配有详细注释的示例代码
- **实践导向**: 每个阶段都有实践项目，学以致用

## 五个学习阶段概述

### 第一阶段：Python 基础 (stage1_basics)

**学习目标**: 掌握 Python 基础语法和编程思维

**主要内容**:
- Python 环境搭建（PyCharm、VSCode）
- 虚拟环境管理
- 变量、数据类型、控制流
- 函数、类与面向对象编程
- 模块、包与文件操作

**实践项目**:
- 计算器程序
- 待办事项列表
- 文件管理器

**预计学习时间**: 2-3 周

---

### 第二阶段：Web 框架与数据库 (stage2_frameworks)

**学习目标**: 掌握主流 Python Web 框架和数据库操作

**主要内容**:
- FastAPI 框架基础与进阶
- Flask 框架基础与进阶
- 路由、请求响应处理
- 数据库集成（SQLAlchemy ORM）
- RESTful API 设计

**实践项目**:
- FastAPI CRUD 应用
- Flask CRUD 应用

**预计学习时间**: 3-4 周

---

### 第三阶段：企业特性 (stage3_enterprise)

**学习目标**: 掌握企业级应用的核心特性

**主要内容**:
- 认证与授权（JWT、OAuth2）
- 基于角色的访问控制（RBAC）
- 中间件开发（日志、错误处理、Request ID）
- 异步任务（Celery + Redis）
- 可观测性（日志、指标、追踪）

**实践项目**:
- 完整的认证授权系统
- 异步任务处理系统

**预计学习时间**: 3-4 周

---

### 第四阶段：测试与 CI/CD (stage4_cicd)

**学习目标**: 掌握自动化测试和持续集成部署

**主要内容**:
- 单元测试与集成测试（pytest）
- 测试夹具与 Mock
- GitHub Actions CI/CD 工作流
- Docker 容器化
- 自动化部署流程

**实践项目**:
- 完整的测试套件
- CI/CD 自动化流水线

**预计学习时间**: 2-3 周

---

### 第五阶段：CRM 实战项目 (stage5_crm_project)

**学习目标**: 通过完整项目整合所有知识点

**主要内容**:
- 完整的 CRM 系统（FastAPI/Flask 双版本）
- 员工、岗位、菜单、角色管理
- RBAC 权限系统
- 工作日志与异步导出
- 前后端分离架构（PureAdminThin）
- 生产级部署配置

**项目特性**:
- 完整的认证授权流程
- 基于角色的权限控制
- 异步任务处理
- 中间件集成
- Docker 容器化部署
- CI/CD 自动化

**预计学习时间**: 4-6 周

## 前置知识要求

### 必备知识
- 基本的计算机操作能力
- 了解文件系统和命令行基础
- 具备基本的英文阅读能力（技术文档）

### 推荐知识（非必需）
- 基础的编程概念（变量、循环、函数）
- HTML/CSS 基础（有助于理解 Web 开发）
- 数据库基础概念

### 零基础学员
如果你完全没有编程经验，不用担心！本课程从零开始，第一阶段会详细讲解所有基础概念。建议：
- 放慢学习节奏，确保理解每个概念
- 多动手实践，运行每一个示例代码
- 完成所有实践项目
- 遇到问题及时查阅故障排除指南

## 学习建议

### 学习方法

1. **循序渐进**: 按照阶段顺序学习，不要跳过基础内容
2. **动手实践**: 运行每一个示例代码，理解其工作原理
3. **完成项目**: 每个阶段的实践项目都要认真完成
4. **查阅文档**: 养成查阅官方文档的习惯
5. **记录笔记**: 记录重要概念和遇到的问题
6. **定期复习**: 使用每个阶段的检查清单进行自我评估

### 学习节奏

**全职学习**: 每天 6-8 小时，预计 3-4 个月完成核心五个阶段

**业余学习**: 每天 2-3 小时，预计 6-8 个月完成核心五个阶段

**周末学习**: 每周 10-15 小时，预计 8-10 个月完成核心五个阶段

### 学习路径建议

1. **新手路径**: 严格按照阶段 1 → 2 → 3 → 4 → 5 的顺序学习
2. **有基础路径**: 如果已掌握 Python 基础，可从第二阶段开始
3. **快速路径**: 有其他语言经验的开发者，可以快速浏览第一阶段，重点学习 Python 特性

### 遇到困难时

- 查阅 `common/troubleshooting.md` 故障排除指南
- 重新阅读相关章节的文档
- 运行示例代码，对比差异
- 使用搜索引擎查找解决方案
- 查阅官方文档和社区资源

## 快速开始

### 1. 环境准备

```bash
# 确认 Python 版本（需要 3.9+）
python --version

# 克隆或下载本课程
cd python_demo
```

### 2. 开始第一阶段

```bash
# 进入第一阶段目录
cd stage1_basics

# 阅读阶段说明
cat README.md

# 开始环境搭建
cd setup
```

### 3. 跟随学习路径

详细的学习路径和阶段依赖关系，请查看 [LEARNING_PATH.md](./LEARNING_PATH.md)
